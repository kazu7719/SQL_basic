### SQLによるデータ操作

## DML
INSERT(データの登録)
UPDATE(データの更新)
DELETE(データの削除)
SELECT(データの検索)

## データの登録

# SELECT文
・・・データの取得の際に使用するSQL文

SELECTでデータを取得するカラムを指定
SELECT <カラム名> FROM <テーブル名>

テーブルに登録されているすべてのカラムを指定

SELECT * FROM <テーブル名>

*・・・すべてのパターンにマッチするものとしてワイルドカードを意味する

# INSERT
・・・テーブルにデータを登録するためのSQL文
INTOという句と組み合わせてINSERT INTOとして使用、設定する値はVALUESという句の後に入力

すべてのカラムに値を入れる場合
INSERT INTO <テーブル名>　VALUES(値1,値2,値3)

特定のカラムのみに値を入れる場合
INSERT INTO テーブル名(カラム名1, カラム名2) VALUES(値1, 値2);


## データの更新

# UPDATE文
・・・データを更新するためのSQL文
使用する際はテーブル名を指定し、SETという句の後に変更内容、WHEREという句のあとに条件を指定

UPDATE <テーブル名>　SET <変更内容>　WHERE <条件>

## データの削除

# DELETE文
・・・データを削除するためのSQL文
テーブル名を指定するためにFROM句を、条件を指定するためにWHERE句を合わせて使用

DELETE FROM <テーブル名>　WHERE <条件>;
